# GameInventoryClient

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.0.4.

# Beschreibung
Game Inventar ist eine Webbasierte Applikation, mit der man seine Gamesammlung managen kann.
In der Applikation kann man Publisher, Genres, Konsolen und Spielmodus erfassen.
Ein Game kann dann beim erfassen der gewünschte Publisher, die vorhandenen Konsolen die Spielmodus und die Genres hinzugefügt werden.
Im Hauptmenu können Spiele danach anhand der Konsolen, der Genres und den Spielmodus sortiert werden.
Der Nutzer kann somit ein Game aus seiner Sammlung direkt suchen und sieht auf welchen Konsolen er das Spiel besitzt und in was für einem Medium (Disc, Digital usw.…)
